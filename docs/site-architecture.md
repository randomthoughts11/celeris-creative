## 2. User journey (primary persona: founder/owner researching agencies)

```
AWARENESS            CONSIDERATION                DECISION            ACTION
Land on hero    →    Scroll storytelling     →    Work / pricing  →   Contact
"Growth isn't        Trusted-by → Services        Case metrics,       Strategy-call form
luck. It's a         → Process → Work →           plans, FAQ          (mailto now,
system."             Why/Stats → AI →             objections          Calendly-ready)
                     Industries → Social proof    handled
```

Conversion paths:
1. **Fast path:** Hero → "Book Strategy Call" (1 click). CTA persists in navbar on every page.
2. **Proof path:** Hero → "View Our Work" → metrics → Final CTA → Contact.
3. **Skeptic path:** Services → pricing transparency → FAQ (objection handling) → Contact.

Every page ends in the Final CTA section; the contact page sets expectations
(diagnosis, written plan, honest recommendation) to lower call anxiety.

## 4. SEO recommendations

**Implemented**
- Per-page titles/descriptions via Metadata API, `metadataBase`, OG + Twitter cards.
- `ProfessionalService` JSON-LD (name, address, email, price range).
- `sitemap.xml` + `robots.txt` generated by Next.
- Google Search Console verification token carried over from the old site.
- Semantic landmarks (`header/main/footer/nav/section` + aria-labels), one `h1` per page.
- All-static prerender (SSG) — fast TTFB, fully crawlable HTML.

**Recommended next**
1. **301 redirects** from old URLs once DNS cuts over: `/about-us → /about`, `/plan → /services`,
   `/portfolio → /work`, `/blog → /` (until blog returns). Configure in `next.config.ts` `redirects()`.
2. Create a real OG image (1200×630) at `src/app/opengraph-image.png` — dark hero frame with wordmark.
3. Launch `/blog` with 2–4 pillar articles ("AI automation for wellness brands", "CRM systems for clinics")
   targeting the wellness/Ayurveda niche where competition is thin.
4. Add per-case-study pages (`/work/[slug]`) with `CaseStudy`/`Review` structured data when
   client permission is secured.
5. Add `FAQPage` JSON-LD on the home page (FAQ content already exists).
6. Keep Google Business Profile (Plano, TX) synced with the site NAP data.

## 5. Performance optimization plan

**Implemented**
- Static generation for every route; zero runtime data fetching.
- No raster images shipped — visuals are CSS gradients, inline SVG, and a canvas particle field
  (capped at 70 particles, devicePixelRatio ≤ 2).
- `next/font` self-hosted fonts with `display: swap` (no layout shift, no third-party font requests).
- Animations are transform/opacity-only (compositor-friendly); marquees are pure CSS;
  scroll animations use GSAP ScrollTrigger with `once: true` where possible.
- `prefers-reduced-motion` disables Lenis, cursor, particles, and reveals.
- Tailwind v4 generates only used utilities; total JS = framework + gsap/framer-motion/lenis.

**When real media is added**
1. Case-study images → `next/image` with AVIF/WebP (already configured), explicit sizes, lazy loading.
2. Hero/CTA background videos → poster image + `preload="none"` + IntersectionObserver play/pause,
   ≤ 2MB AV1/H.265, no autoplay on mobile data-saver.
3. Video testimonials → façade pattern (thumbnail, load player on click).
4. Monitor CWV in the field via Vercel Analytics or CrUX; budget: LCP < 2.0s, INP < 200ms, CLS < 0.05.
5. If GSAP bundle weight becomes a concern, lazy-load section animations with dynamic imports.

## 6. Copywriting system

Voice: confident, concise, numbers-first, allergic to agency clichés.
- Headlines state a belief ("Growth isn't luck. It's a system.").
- Every section earns its place in the conversion argument (see user journey).
- CTAs are commitments, not commands: "Book Strategy Call" backed by "no pitch deck,
  written plan in 48 hours, honest recommendation".
- Positioning: *premium strategic growth partner* across AI automation, digital marketing,
  creative design, branding, web development, CRM, content systems, podcast production,
  lead generation, and business growth.

**Note on placeholder facts:** stats (3.2x ROAS, 92%, 14 days, 40+ brands), case studies,
testimonials, and client names are illustrative placeholders written to show the copy system.
Replace with verified client data in `src/lib/data.ts` before launch.
